# 🕌 Athan Daily

A modern Islamic companion for Muslims around the world.

Athan Daily provides accurate prayer times, Qibla direction, nearby mosques, Hijri calendar, and prayer reminders in a clean and simple interface.

---

## Features

- 🕌 Accurate Prayer Times
- 🧭 Qibla Compass
- 📍 Nearby Mosques
- 📅 Hijri Calendar
- 🔔 Prayer Notifications
- 🌎 Automatic Location Detection
- ⚡ Fast & Lightweight
